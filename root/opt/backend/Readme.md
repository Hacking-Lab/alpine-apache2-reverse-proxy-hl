# Backend Service
This content is being delivered by the `backend` service. This service is `behind` the reverse proxy (Apache & PHP)

```
            +------------+                +---------------+
            | Apache     |                | Apache + PHP  |
            | Reverse    |                | Backend       |
  +-------->+ Proxy      +--------------->+ Service       |
            |            |                |               |
            +------------+                +---------------+

```
